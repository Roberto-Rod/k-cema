This folder must contain two "Touchstone" (.s2p) files in it for the calibrated path between UUT and RF Signal Generator,
and for the calibrated path between UUT and RF Spectrum Analyser. The calibration files must cover a minimum frequency range
of 400-8000 MHz.

The two files must be named:

uut_to_signal_generator.s2p
uut_to_spectrum_analyser.s2p

These files must be in "DB" or "MA" format.
